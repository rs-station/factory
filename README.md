## Development

To set up the development environment:

```bash
# Clone the repository
git clone https://github.com/yourusername/factory.git
cd factory

# Install development dependencies
pip install -e ".[test,docs]"

# Run tests
pytest

# Build documentation
cd docs
make html
```
